MBS Hosting Service
by Green Flamingo
Project Description
Migrant Business Show (MBS) is a Web application service used for hosting and managing fair-type events. It utilizes Java EE Spring framework that allows efficient integration of event related data with the website of the host. It aims to provide the end user with information about events such as open days, speakers, exhibitors, press items and much more. Multiple user access is enabled with the power of Java spring security. All users have their own dashboard that allows them to access and edit relevant content. Additionally, the MBS web application includes a separate component developed for the purpose of organizing competitions related to events. It includes features such as nomination, voting and monitoring.


Technologies used:
  • Java EE
  • Thymeleaf
  • HTML/CSS
  • Javascript
  • Tomcat
  • CentOS linux distribution
  • InfusionSoft API
